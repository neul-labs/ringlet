# clown-cli

CLI orchestrator for coding agents.

## Installation

```bash
pip install clown-cli
```

## Usage

```bash
clown --help
```

## Features

- Manage multiple coding agent profiles
- Support for Claude Code, Grok, Codex, Droid, OpenCode, and more
- Provider abstraction for Anthropic, OpenAI, OpenRouter, and custom backends
- Event-driven hooks for integration
- Proxy integration with intelligent request routing
